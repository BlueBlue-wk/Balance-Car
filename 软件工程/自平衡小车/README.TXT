实验器材:
战舰V3&精英STM32F103开发板

实验目的:
学习ATK- BLE01蓝牙串口模块的使用。

硬件资源:
1, DS0、DS1(连接在PB5、PE5)
2, 串口1(波特率: 115200, PA9 / PA10连接在板载CH340上)
3, 按键KEY0(PE4) / KEY1(PE3) / KEY2(PE2) / KEY_UP(PA0, 也称之为WK_UP)
4, ATK-BLE01模块1个.
5, 手机一台（Android4.3或iphone4s以上版本的手机）

ATK-BLE01      战舰V3&精英STM32F103开发板
VCC < ------------ > 5V
GND < ------------ > GND
TXD < ------------ > PB11
RXD < ------------ > PB10
STA < ------------ > PA4
WKUP < ----------- > PA15

软件资源：
手机APP使用“LightBlue”，软件安装包在软件文件夹内


实验现象:
手机通信，实现手机端与板载开发板的ATK - BLE01模块通讯。
手机APP连接模块，连接成功信息会通过串口输出，手机端发送“+LEDD0:ON”字符串，板上的LED0灯点亮，
发送“+LED0:OFF”字符串，板上的LED0灯熄灭。按下开发板KEY1按键，发送数据给手机端显示，同时手机端发送的数据会通过串口输出。
